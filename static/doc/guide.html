1. Présentation de l'application

NutriProof est une application de bureau conçue pour extraire automatiquement les données de bilans lipidiques et résultats de laboratoire à partir de documents PDF numérisés.

L'application utilise la reconnaissance optique de caractères (OCR) via Tesseract pour identifier les paramètres et leurs valeurs, puis permet une révision manuelle avant l'exportation en fichier Excel.

Fonctionnalités principales

  • Téléversement de PDF — Importation de documents numérisés multi-pages
  • Extraction automatique — Détection des paramètres et des valeurs par OCR
  • Visualisation interactive — Affichage des zones détectées (bounding boxes) superposées sur l'image
  • Correction manuelle — Édition directe des valeurs extraites dans un tableau
  • Double validation — Processus de vérification en deux étapes (V1 / V2)
  • Export Excel — Téléchargement des données structurées au format .xls

4. Traitement OCR

Le traitement OCR s'exécute automatiquement après le téléversement du PDF. Chaque page est traitée séquentiellement.

Suivi de la progression

Pendant le traitement, la colonne Statut affiche la progression en temps réel :

  • 1/5 pages — Traitement en cours (progression page par page)
  • Terminé — Traitement complété avec succès
  • Annulé — Traitement annulé par l'utilisateur

Annuler un traitement

Si le document est volumineux, vous pouvez annuler le traitement en cours en cliquant sur le bouton qui apparaît à droite du statut pendant le traitement.

Attention : L'annulation arrête le traitement à la page en cours. Les pages déjà traitées restent disponibles pour la révision.

5. Révision et correction

Cliquez sur le bouton d'un document pour ouvrir la page de révision.

Interface de révision

L'écran se divise en deux panneaux :

PanneauContenu
Gauche Tableau éditable des paramètres et valeurs extraites, suivi des contrôles de pagination
Droite Image de la page PDF avec les zones de détection (bounding boxes) superposées

Zones de détection (bounding boxes)

Les zones détectées par l'OCR sont affichées par-dessus l'image de la page :

  • Rectangles colorés pleins — Zones associées aux paramètres configurés. Chaque paramètre possède sa propre couleur (label et valeur).
  • Rectangles bleus fins — Tous les blocs de texte détectés par Tesseract (mode débogage).

Survolez une zone avec la souris pour afficher une infobulle indiquant le paramètre et sa valeur.

Modifier les valeurs extraites

1
Dans le tableau de gauche, cliquez sur la cellule Valeur du paramètre à corriger.
2
Saisissez la valeur correcte. La sauvegarde est automatique à chaque modification.
Astuce : Comparez visuellement la valeur du tableau avec la zone colorée correspondante sur l'image pour vérifier l'exactitude de l'extraction.

Navigation entre les pages

Les contrôles de pagination se trouvent sous le tableau :

  • — Page précédente

  • Champ numérique — Saisissez un numéro de page et appuyez sur Entrée pour y accéder directement

6. Validation

Le processus de validation se fait en deux étapes directement depuis la page principale :

CaseObjectif
V1 Première vérification — le réviseur confirme que les données sont correctes
V2 Seconde vérification — un second réviseur (ou le même) confirme la validation

Cochez la case correspondante pour chaque document vérifié. L'état est sauvegardé automatiquement.

9. Dépannage

Je ne peux pas me connecter

  • Vérifiez que votre adresse courriel et mot de passe sont corrects.
  • Utilisez la fonction « Mot de passe oublié » pour réinitialiser votre mot de passe.
  • Vérifiez votre connexion Internet (nécessaire pour l'authentification Firebase).

L'OCR ne détecte pas certaines lignes

  • Assurez-vous que le PDF est numérisé avec une résolution suffisante (300 DPI recommandé).
  • Les documents flous, inclinés ou de mauvaise qualité peuvent affecter la détection.
  • Les textes très petits ou isolés sont plus difficiles à détecter.

Les valeurs extraites sont incorrectes

  • Utilisez la page de révision pour corriger manuellement les valeurs.
  • Comparez visuellement les zones colorées avec les données du tableau.
  • Les corrections sont sauvegardées automatiquement.

Le traitement est lent sur un gros document

  • Chaque page est traitée séquentiellement. Un document de 20 pages prendra nécessairement plus de temps.
  • Vous pouvez annuler le traitement en cours si nécessaire et le relancer plus tard.

L'application ne répond plus

  • Fermez l'application et relancez-la.
  • Si le problème persiste, vérifiez que le processus Python n'est pas bloqué (le redémarrage de l'application devrait régler le problème).
