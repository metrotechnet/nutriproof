# NutriProof Desktop

Ce dossier contient l'intégration Electron pour lancer l'application Flask en mode bureau.

## Installation

1. Installe Node.js et npm.
2. Dans ce dossier, lance :
   npm install

## Lancement

1. Assure-toi que les dépendances Python sont installées (voir README principal).
2. Depuis ce dossier :
   npm start

L'application démarre le serveur Flask et ouvre la fenêtre Electron sur http://localhost:8080.
